# :robot: RPA-Starter

### RPA?
    RPA(Robotic Process Automation)란 무엇일까?
    RPA는 규칙 기반의 반복적인 업무를 소프트웨어 로봇이 대신 처리하는 디지털 업무 자동화 솔루션으로
    즉, 사람이 PC에서 작업하는 단순 반복적 업무방식을 모방하여 사람이 수행하는 것과 동일한 방식으로 작업을 수행하는 소프트웨어다.
